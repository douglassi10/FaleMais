# TelzirFaleMais
Trata-se de uma aplicação web que disponibiliza com antecedência o preço das ligações que podem ser realizadas pelo plano FaleMais.

# Tecnologias Utilizadas
- JavaScript
- Html 5
- CSS
- Bootstrap 5
